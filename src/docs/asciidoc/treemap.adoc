== Introduction

The DocOps Treemap Extension allows you to create hierarchical data visualizations that show proportional relationships through nested rectangles. Treemaps are excellent for displaying budget allocations, disk space usage, portfolio compositions, and any data where you want to show both hierarchy and proportion.

This guide provides examples of different treemap styles and use cases.

== Basic Concepts

=== What are Treemaps?

Treemaps visualize hierarchical data using nested rectangles. Each rectangle's area is proportional to its value, making it easy to:

* Compare relative sizes at a glance
* Identify the largest contributors
* Understand resource distribution
* Spot patterns in hierarchical data

==== Data Format

Each data row contains pipe-separated values:

[source]
----
Label | Value | Description | Color | Metric
----

* **Label** (required): Category name
* **Value** (required): Numeric value for sizing
* **Description** (optional): Additional context
* **Color** (optional): Custom hex color
* **Metric** (optional): Formatted display value

== Best Practices

=== Data Guidelines

1. **Reasonable Range** - Use 4-10 categories for optimal readability
2. **Meaningful Labels** - Keep category names concise but descriptive
3. **Proportional Values** - Large differences in values work better than similar values
4. **Context Matters** - Use descriptions to add context without cluttering

[.back-to-top]
<<top, ðŸ”>>

=== Visual Design

1. **Color Consistency** - Choose palettes that match your brand or content
2. **Text Readability** - Ensure labels are visible against background colors
3. **Appropriate Size** - Larger treemaps show more detail
4. **Theme Selection** - Match theme to your presentation context

[.back-to-top]

=== Use Cases

**Best for:**

* Budget allocations
* Resource distribution
* Market share analysis
* Disk space usage
* Time allocation
* Portfolio composition

**Not ideal for:**

* Time-series data (use line charts)
* Precise comparisons (use bar charts)
* Many small categories (use pie charts)

DocOps Treemaps provide a powerful way to visualize hierarchical proportional data. With multiple themes, color palettes, and configuration options, you can create compelling visualizations that communicate complex information at a glance.
