== DocOps Extensions

DocOps Extensions can now be embedded in Asciidoctor Files. Using the DocOps converter, you can seamlessly integrate visualizations, diagrams, and interactive components directly into your Asciidoctor documents. This allows for a more engaging and informative way to present information, making it easier for readers to understand complex concepts and processes.
