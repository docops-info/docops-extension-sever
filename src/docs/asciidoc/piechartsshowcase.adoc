= DocOps PieCharts Showcase

[abstract]
The purpose of this document is to showcase the DocOps Pie Chart visualization capabilities.
It provides a gallery of example pie charts and multi-pie charts generated from simple data files using the docops macro.
Use it as a reference to:

- understand different pie chart styles available and how they render in light and dark modes
- see how to enable interactive controls and supply data from external files
- copy working snippets when authoring your own pie chart visualizations in documentation
