= Combination Charts

[[top]]
== Introduction

Combination charts layer bars and lines on one chart, perfect for showing related metrics with different scales. Track revenue (bars) against growth rate (line), or sales volume (bars) versus profit margin (line). Dual Y-axes keep everything readable.

=== Key Features

* *Mix bars and lines* - Combine chart types for richer insights
* *Dual Y-axes* - Show metrics with different scales side-by-side
* *Smart legends* - Shows chart type for each series
* *Smooth or angular lines* - Visual control over line appearance
* *Auto-scaling* - Each Y-axis scales independently

== Common Use Cases

* *Financial analysis* - Revenue (bars) vs. growth rate or margin (line)
* *Operational metrics* - Volume (bars) vs. efficiency or quality (line)
* *Marketing dashboards* - Spend (bars) vs. ROI or conversion rate (line)
* *Product analytics* - Usage (bars) vs. satisfaction score (line)
* *Budget tracking* - Actual spend (bars) vs. cumulative % of budget (line)

== Format Options

=== Basic Format

[source,asciidoc]
....
[docops,combination]
----
title=Chart Title
xLabel=X-Axis Label
yLabel=Primary Y-Axis Label
yLabelSecondary=Secondary Y-Axis Label
dualYAxis=true
smoothLines=true
showGrid=true
---
Series Name | Type | X-Value | Y-Value | Color | Y-Axis
Revenue | BAR | Q1 | 50000 | #3498db | PRIMARY
Growth Rate | LINE | Q1 | 15.5 | #e74c3c | SECONDARY
----
....

=== Common Options

* `dualYAxis=true` - Enable second Y-axis for different scales
* `smoothLines=true` - Curved lines (default: false)
* `showGrid=true` - Display background grid (default: true)
* `showLegend=true` - Display legend (default: true)
* `showPoints=true` - Show data point markers on lines (default: true)
* `useDark=true` - Dark theme

=== Data Columns

* `Series Name` - Label shown in legend
* `Type` - BAR or LINE
* `X-Value` - Category or time period label
* `Y-Value` - Numeric value
* `Color` - Hex color (e.g., #3498db)
* `Y-Axis` - PRIMARY or SECONDARY

== Best Practices

* *Use dual Y-axes for different units* - Revenue ($) on left, rate (%) on right
* *Bars for absolutes, lines for rates* - Volume as bars, efficiency as lines
* *Match colors to meaning* - Blue for revenue, red for costs, green for profit
* *Limit to 2-4 series* - One bar series, 1-3 line series maximum
* *Label both Y-axes clearly* - Include units in axis labels

[TIP]
====
Combination charts excel when one metric drives the other (e.g., marketing spend drives conversions). Use bars for the driver (input) and lines for the outcome (result) to show cause-and-effect visually.
====
