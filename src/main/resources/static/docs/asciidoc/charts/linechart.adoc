= Line Charts

[[top]]
== Introduction

Line charts reveal trends over time. Whether tracking daily metrics, monthly performance, or quarterly growth, line charts make patterns, peaks, and valleys immediately visible. Multiple series let you compare related trends side-by-side.

=== Key Features

* *Multi-series support* - Compare up to 6 trends on one chart
* *Smooth or angular lines* - Choose the visual style that fits your data
* *Auto-scaled axes* - Smart axis ranges based on your data
* *Interactive legend* - Toggle series visibility
* *Area fill option* - Emphasize volume under the curve

== Common Use Cases

* *Performance tracking* - Monitor KPIs over weeks, months, or quarters
* *Trend analysis* - Identify seasonal patterns or growth trajectories
* *Before/after comparisons* - Show impact of changes or interventions
* *Forecasting visualization* - Display historical data with projected values
* *Multi-metric dashboards* - Compare related trends (e.g., sales vs. costs)

== Format Options

=== Basic Format

[source,asciidoc]
....
[docops,line]
----
title=Chart Title
width=800
smoothLines=false
showArea=true
useDark=false
---
Series 1 | Jan | 40
Series 1 | Feb | 70
Series 2 | Jan | 22
Series 2 | Feb | 33
----
....

=== Common Options

* `smoothLines=true` - Curved lines (default: false for angular)
* `showArea=true` - Fill area under lines (default: true)
* `useDark=true` - Dark theme
* `width=800` - Chart width (default: 900)
* `backgroundColor=#F5F5F5` - Custom background color

== Best Practices

* *Use consistent time intervals* - Don't skip months or mix weekly/monthly data
* *Limit to 4-6 series* - More lines create visual confusion
* *Choose smooth lines for trends* - Angular lines for exact data points
* *Start Y-axis at zero* - Unless showing small variations in large numbers
* *Label key inflection points* - Annotate major changes or milestones

[TIP]
====
For volatile data (daily stock prices), use smooth lines to show overall trend. For discrete measurements (monthly sales), use angular lines to represent actual data points accurately.
====
