Pie Charts

1. Introduction

Pie charts show proportions of a whole at a glance. Perfect for market share, budget breakdowns, or any data where the parts sum to 100%. The modern donut style adds visual polish while maintaining clarity.

1.1. Key Features

  • Classic pie or modern donut - Choose the style that fits your aesthetic

  • Auto percentages - Labels show both values and calculated percentages

  • Interactive legend - Hover effects for engagement

  • Glass morphism design - Modern, polished appearance

  • Dark mode support - Seamless light/dark theme switching

4. Common Use Cases

  • Market share analysis - Show competitive positioning by segment

  • Budget breakdowns - Visualize spending allocation across departments

  • Survey results - Display response distribution for single-choice questions

  • Resource allocation - Show headcount or asset distribution

  • Portfolio composition - Illustrate investment or product mix

🔝

5. Format Options

5.1. Basic Format

[docops,pieslice]
----
title=Chart Title
legend=true
donut=true
percentages=true
---
Category 1 | 30
Category 2 | 25
Category 3 | 20
Category 4 | 15
Category 5 | 10
----

5.2. Common Options

  • donut=true - Modern donut style (default: false for classic pie)

  • legend=true - Show interactive legend with percentages (default: true)

  • percentages=true - Display % labels on slices (default: true)

  • hover=true - Enable hover effects (default: true)

  • useDark=true - Dark theme

  • width=600 - Chart width (default: 500)

  • height=600 - Chart height (default: 500)

5.3. Custom Colors

Add color as third column:

Category 1 | 30 | #3498db
Category 2 | 25 | #e74c3c

🔝

6. Best Practices

  • Limit slices to 5-7 - More slices become hard to distinguish visually

  • Order by size - Place largest slice at 12 o’clock, descend clockwise

  • Use donut for modern docs - The donut style looks more polished and less dated

  • Show percentages for small slices - Values under 10% are hard to judge by area alone

  • Avoid pie charts for time series - Use line or bar charts for trends over time

If you have more than 7 categories, group smaller slices into an "Other" category. Pie charts lose effectiveness when slices become too thin to label clearly.

7. 🥧 Multi-Pie Display

Compact percentage indicators for dashboards and summaries

Key features:

  • Compact radial percentage displays

  • Multiple pies in horizontal layout

  • Custom base and outline colors

  • Minimal space footprint

  • Perfect for dashboards and summary cards

  • Dashboard metrics - Compact KPI indicators in summary views

---
Label | Value
Label | Value
----
Option Purpose

baseColor

Background circle color (hex)

outlineColor

Filled arc color (hex)

scale

Size multiplier (default: 1.0)

useDark=true

Dark theme

🔝

7.5. Best Practices

  • Limit to 4-6 pies - More pies become cluttered horizontally

  • Use semantic colors - Green outline for success, red for warnings

  • Keep labels short - 1-2 words for clean layout

  • Scale for context - 0.8 for cards, 1.2 for hero metrics

  • Group related metrics - Similar categories (all performance, all quarterly)

  • Show percentages only - Values should be 0-100 for radial display

For dashboards: Use scale=0.9 with consistent baseColor/outlineColor across all metrics. This creates a unified visual language where users instantly recognize "green outline = healthy, red outline = attention needed."
