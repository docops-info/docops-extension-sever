1. Introduction

The DocOps Treemap Extension allows you to create hierarchical data visualizations that show proportional relationships through nested rectangles. Treemaps are excellent for displaying budget allocations, disk space usage, portfolio compositions, and any data where you want to show both hierarchy and proportion.

This guide provides examples of different treemap styles and use cases.

2. Basic Concepts

2.1. What are Treemaps?

Treemaps visualize hierarchical data using nested rectangles. Each rectangle’s area is proportional to its value, making it easy to:

  • Compare relative sizes at a glance

  • Identify the largest contributors

  • Understand resource distribution

  • Spot patterns in hierarchical data
