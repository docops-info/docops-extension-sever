1. Introduction

Create beautiful isometric step infographics to visualize processes, workflows, and progress

Key features:

  • Isometric 3D step visualization with numbered stages

  • Configurable themes and color palettes

  • CSV-style data format with header row for easy authoring

  • Support for custom colors per step

  • Adjustable canvas size, layout offsets, and label positioning

5. Format Options

5.1. Basic Structure

[docops,steps,controls=true]
----
infographic=isometric-steps
version=1
title=My Steps Title
subtitle=Optional subtitle text
canvasWidth=1220
canvasHeight=760
theme=classic
palette=vibrant
---
Order | Title | Description | Color
1 | Step One | Description of step one | #6EAEFF
2 | Step Two | Description of step two | #69DEE5
----

5.3. Step Data Columns

Column Required Purpose

Order

Yes

Numeric order of the step (must be unique)

Title

Yes

Short title for the step

Description

No

Brief description text

Color

No

Custom hex color (e.g., #6EAEFF)

5.4. Data Format Rules

  • Config section and data section are separated by ---

  • First row after --- is the header row

  • Columns are separated by | (pipe)

  • Order values must be unique integers

  • Color is optional; palette colors are used when omitted

Example:

infographic=isometric-steps
version=1
title=My Process
---
Order | Title | Description | Color
1 | First Step | Getting started | #6EAEFF
2 | Second Step | Making progress | #69DEE5
3 | Final Step | Completing the task | #A7E58A

6. Best Practices

  • Keep titles short - Use 2-3 words per step for best visual results

  • Limit to 3-8 steps - Too many steps reduce readability

  • Use consistent colors - Stick to a palette or let the theme handle it

  • Write concise descriptions - One line per step works best

  • Order sequentially - Use 1, 2, 3…​ for clear progression

  • Choose meaningful subtitles - They provide context for the entire infographic

For best results, keep step titles short and action-oriented. Use the description field for additional context, and let the color palette create visual harmony unless you need specific brand colors.
