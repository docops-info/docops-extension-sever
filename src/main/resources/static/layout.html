Usage Examples

Bar Chart Example

Create a responsive bar chart with custom styling

JavaScript 
// Initialize DataViz
const chart = new DataViz('#chart-container', {
  type: 'bar',
  responsive: true,
  theme: 'modern'
});

// Add data
chart.setData([
  { label: 'Q1', value: 120 },
  { label: 'Q2', value: 180 },
  { label: 'Q3', value: 240 },
  { label: 'Q4', value: 200 }
]);

// Customize appearance
chart.setOptions({
  colors: ['#3B82F6', '#10B981', '#F59E0B', '#EF4444'],
  animation: 'slideUp',
  showValues: true
});

// Render the chart
chart.render();

Real-time Line Chart

Live data updates with smooth animations

JavaScript 
// Create real-time line chart
const lineChart = new DataViz('#line-chart', {
  type: 'line',
  realTime: true,
  maxDataPoints: 50
});

// Initial data
const timeData = generateTimeSeriesData();
lineChart.setData(timeData);

// Configure real-time updates
lineChart.setOptions({
  updateInterval: 1000,
  smoothTransitions: true,
  showGrid: true,
  strokeWidth: 3
});

// Start real-time updates
setInterval(() => {
  lineChart.addDataPoint({
    x: new Date(),
    y: Math.random() * 100
  });
}, 1000);

lineChart.render();

Dashboard Integration

Multiple charts in a responsive dashboard layout

JavaScript 
// Create dashboard with multiple charts
const dashboard = new DataViz.Dashboard('#dashboard');

// Add multiple visualizations
dashboard.addChart('sales-chart', {
  type: 'bar',
  title: 'Monthly Sales',
  data: salesData,
  container: '.sales-container'
});

dashboard.addChart('users-chart', {
  type: 'line',
  title: 'User Growth',
  data: userGrowthData,
  container: '.users-container'
});

dashboard.addChart('distribution-chart', {
  type: 'donut',
  title: 'Traffic Sources',
  data: trafficData,
  container: '.traffic-container'
});

// Sync interactions across charts
dashboard.enableCrosshairs();
dashboard.linkTimeRanges();

// Auto-refresh every 5 minutes
dashboard.setAutoRefresh(300000);

Quick Start

1. Installation

npm
npm install dataviz-pro

Or include via CDN:

<script src="https://cdn.dataviz-pro.com/v2.1.0/dataviz.min.js"></script>

2. Basic Usage

// Import the library
import DataViz from 'dataviz-pro';

// Create your first chart
const chart = new DataViz('#my-chart', {
  type: 'bar',
  data: [
    { label: 'Jan', value: 100 },
    { label: 'Feb', value: 150 },
    { label: 'Mar', value: 200 }
  ]
});

chart.render();
